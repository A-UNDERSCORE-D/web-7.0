you can view different branches current state on this page by reaching 

http://freenode.github.io/web-7.0/BRANCHNAME/
Simply replace the BRANCHNAME with the branch you want to see, for example http://freenode.github.io/master/

This does also work for Pull Requests except if they are made from a fork. use 

http://freenode.github.io/web-7.0/pull-N/
replace N with the id of the pull request as can be obtained on GitHub
